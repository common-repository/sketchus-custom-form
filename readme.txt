Now you can easily create WordPress Custom contact form plugin. Simple to use and flexible enough to adapt changes.

== Description ==

Sketchus Custom Form has the ability to manage multiple contact forms, as an add-on you can also customize the email content flexibly using some simple markup. This form support reCaptcha, Akismet spam filter, strong Ajax-powered submission and lots more.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1.) Upload the complete folder of `sketchus-custom-form` to the `/wp-content/plugins/ directory`. 
2.) Go to ‘Plugins’ menu and activate this plugin in WordPress. 

Now, you would notice a `Custom Forms` menu in our WordPress admin panel. 

For general usage instructions, you can have a glance at [plugin website](http://dev.sketchus.com/custom-form/).
